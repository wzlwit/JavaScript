Assignment
As a user, I should be able to select an option from following through variable 'functionToOperate':

    search a student by name and know his index location in an array
    add a new student
    remove a student
    display all students
    display all students sorted by name
    display a random student's name
    take the third letter of every student's name, append it to the beginning of their name and display the sorted list of students 

 
Regarding sorting: The list should be sorted alphabetically irrespective of the case of the name.
For example, Ajohn, Lorem, Ipsum, foo, bar, list of students should be sorted as:
 
-> Ajohn, , bar, foo, Ipsum, Lorem
